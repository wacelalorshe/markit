 متجر إلكتروني مصغر

مشروع متجر إلكتروني مصغر مع لوحة تحكم لإدارة المنتجات.

 المميزات

- عرض المنتجات في واجهة المتجر
- لوحة تحكم لإضافة وحذف المنتجات
- تصميم متجاوب
- سهولة الاستخدام

 كيفية الاستخدام

1. افتح `index.html` لعرض المتجر
2. افتح `admin.html` للوصول إلى لوحة التحكم
3. استخدم لوحة التحكم لإضافة منتجات جديدة أو حذف المنتجات الحالية

 التكنولوجيا المستخدمة

- HTML
- CSS
- JavaScript

 الرخصة

هذا المشروع مرخص تحت رخصة MIT.
